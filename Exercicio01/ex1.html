  • Github e Git Bash

    • No Github nós guardamos nosso projetos feitos com diversos tipos de linguagens de desenvolvimento, utilizamos o terminal Git Bash para enviar tais projetos.

    • É uma maneira da apresentar estilo aos códigos criado.
